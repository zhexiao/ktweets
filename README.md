<a name="Installation"/>
- VirtualEnv
```shell
    $ virtualenv -p python3 env
```

- Django 1.9.2
```shell
    $ pip install django
    $ django-admin startproject kweets
```

- Mysql
```shell
    $ sudo apt-get install python3-dev libmysqlclient-dev
    $ pip install mysqlclient
```
